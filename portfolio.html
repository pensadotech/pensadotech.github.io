Portfolio

Word games

The game follows the basic idea of the “hangman game” but without anyone getting hanged. It is a friendly game where the player needs to guess the hidden word revealing one letter a time, and up to 12 guesses.

If you are a developer, visit the code sources stored in the Github repo.

RPG Star wars

The game is a combat driven by random health points and random attack and counter attack powers. The player selects a character, and must face the remaining opponents to win the battle. By selecting one enemy at a time and using the attack button, the faith of the battle is left to chances.

If you are a developer, visit the code sources stored in the Github repo.

Trivia Night (solar system)

The game will test the player knowledge about our solar system by presenting five question that must be answered in 40 seconds. The game will celebrate with an answer is correct, and give feedback when selecting the wrong answer.

If you are a developer, visit the code sources stored in the Github repo.

The New York times API

The application connects to The New York Times to provide the latest top stories or five the user the capability to search special topics. The provided abstract will give a link to access the full story.

If you are a developer, visit the code sources stored in the Github repo.
